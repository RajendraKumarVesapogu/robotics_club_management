Overview

The Robotics Club Management System is designed to enhance collaboration among robotics enthusiasts, streamline communication, and foster an environment of innovation. With a user-friendly interface, the CMS allows members to easily navigate through their profiles, manage group activities, and participate in club events.

This system not only serves as a database for member information but also provides tools for group management, ensuring that every team has the guidance of experienced mentors. Whether you are an alumni, staff member, mentor, or admin, this platform is tailored to meet your specific needs and roles within the club.

Key Features

User Authentication

Secure login for members with various roles, including Alumni, Staff, Mentor, and Admin, ensuring that only authorized users can access sensitive information.

Profile Pages

Each user has a dedicated profile showcasing their name, role, group affiliation, account creation date, and contributions to the club.

Group Management

Flexibility in organization with groups requiring assigned mentors, enabling effective supervision and guidance for all members.

User Data Management

Effortlessly save usernames, roles, and groups to a file for streamlined management and reporting, simplifying the administrative tasks.

Intuitive Interface

Enjoy a user-friendly experience built with Django and enhanced CSS, making navigation and usage straightforward for all members.

Technologies Used

  • Django: A high-level Python web framework for rapid development and clean design.
  • SQLite: The default database used for storing user and group data.
  • Python: The programming language powering our backend development.
  • HTML/CSS/Bootstrap: For crafting a responsive and engaging frontend.
